Before You Write In

Most "how do I…" questions are covered in the Features Guide. It walks through every panel, menu, and workflow with examples. For bug reports, please include:

  • macOS version (About This Mac → More Info → macOS).
  • Terminal Conductor version (Terminal Conductor menu → About).
  • What you did, what you expected, what happened instead.
  • Any red entries in the Console tab of View → Toggle Developer Tools.
  • If the App crashed: the most recent crash log from ~/Library/Logs/DiagnosticReports/.

Frequently Asked Questions

General

What is Terminal Conductor?

Terminal Conductor is a macOS application that wraps your terminal sessions with identity, context, memory, and orchestration. It's built for developers running multiple AI coding agents alongside shells, containers, and SSH sessions.

Does it replace my existing shell or dotfiles?

No. Terminal Conductor spawns real shells using your existing configuration (zsh, bash, fish — whatever your $SHELL is). Your aliases, prompts, plugins, and tools behave exactly as they always have.

What macOS versions are supported?

macOS 14.0 (Sonoma) or later. Apple Silicon and Intel Macs are both supported.

Is any data sent to the cloud?

No. Terminal Conductor has zero telemetry and no external servers. All state lives locally. See the Privacy Policy for the full breakdown.

Tabs, groups, and workspaces

How do I create a tab?

Click the + button in the tab bar, or press ⌘T. Click the caret next to it to pick a Pinned or Recent connection instead of a local shell.

How do I group tabs?

Right-click a tab → Add to group → pick an existing group or create a new one. Grouped tabs render with a colored bar above them and can be acted on together (broadcast, briefing, etc.).

How do I split a tab into multiple columns?

Right-click the tab → Split view → choose 2–5 columns. Press Esc to collapse.

Tabs are auto-named "Tab A", "Tab B" — how do I rename?

Click the tab's color dot or long-press the tab; a label editor appears. Once a label is set, it takes priority over the auto-name.

AI agents

Which AI agents does Terminal Conductor support?

Any that run in a terminal. Auto-detection is built for Claude Code, Codex CLI, Gemini CLI, and Aider, including model-name recognition where the agent exposes it. Other CLI-based agents still work — you just won't get the badge.

What is @tab:?

Type @tab: in any terminal to open a picker listing other tabs. Select one and Terminal Conductor injects a formatted context block — identity, recent output, activity — into your current prompt. @group: does the same for whole groups.

How does /fork work?

In any agent tab, type /fork. Terminal Conductor captures the resume command, creates a paired child tab, and enters the resume command in both — you now have two tabs resuming the same conversation, diverging from there. Works for Claude Code and Codex out of the box; for Gemini we built our own fork flow because the official CLI didn't ship one.

Claude says "transcript is too large to branch" — can I still fork?

Yes. Terminal Conductor forces the fork in this case — we capture the resume command directly and start a parallel tab regardless of Claude's in-TUI check.

Sessions and persistence

What happens when I quit the App?

A lightweight session daemon keeps your PTYs alive in the background, so running processes continue. Relaunching Terminal Conductor restores every tab exactly where you left it.

I accidentally closed a tab — can I get it back?

If Settings → Terminal → Keep shell on tab close is on, closing a tab detaches it rather than killing it. Click the caret next to the + button to see detached sessions and reattach.

My screen went blank after opening many tabs — what do I do?

Force-quit Terminal Conductor (⌘⌥Esc → Terminal Conductor → Force Quit) and relaunch. The daemon preserves your PTYs, so your tabs and running agents will come back. If the blank-screen issue repeats, open View → Toggle Developer Tools and send us what's in the Console.

Connections

How do I pin an SSH host?

Click the caret next to the + button → Recent tab → hover a row → click the pin icon. Pinned entries never evict from the list.

Does Terminal Conductor type my SSH passwords?

Only if you enable that feature. Turn on Settings → Security → SSH auto-login, enter the password once, and Terminal Conductor will auto-type it when it sees a matching password prompt. Passwords are stored with light obfuscation in localStorage — this is convenience, not security. Use keys for anything that matters.

Can I paste images into a container or SSH session?

Yes. Copy an image, switch to the target tab, press ⌘V. Terminal Conductor writes it to a temp file, transfers it to the right environment (local path, docker cp, or scp for Tailscale-authenticated SSH), and pastes the path into your prompt.

Visual modes and borders

How do I change visual mode or border?

Open the right panel → Appearance → Visual Mode / Border Decoration. Settings are per-tab by default; use groups to apply to several at once.

How do I turn off the JARVIS HUD border?

The JARVIS HUD is bound to the JARVISmaxxing visual mode. Switch to a different visual mode and the border changes with it. Or turn off Force mode borders in Settings → Appearance to decouple mode and border.

Can I use the 1992 skin on a modern macOS?

Yes — it's purely cosmetic. All functionality is identical. Turn it on via Settings → Appearance → Skin.

Privacy and data

Where is my data stored?

~/Library/Application Support/TerminalConductor/ (session database, scrollback, persisted state) and ~/Library/Preferences/ (app settings). Nothing leaves your Mac.

How do I wipe all Terminal Conductor data?

Quit the App, then delete the directories in the previous answer. See the Privacy Policy, Section 7, for the full step-by-step.

Terminal Conductor SSH

A visual intelligence layer for your remote terminal sessions.

Purpose-built for developers running AI coding agents over SSH.

One window. Every remote session. Perfect harmony.

Terminal Conductor SSH wraps your SSH tabs in identity, context, and memory. Every tab is visually distinct. Every agent is auto-detected. Every session survives an app restart. No sprawl, no sprawl anxiety, no lost work.

macOS 14+ Apple Silicon & Intel SSH-only Zero telemetry Sandboxed

What it does

Terminal Conductor SSH is a single-window Mac app that lets you manage many SSH sessions as named, color-coded, persistent tabs — with a bird's-eye workspace overview, AI agent awareness, remote environment probing, and cross-tab context sharing built in.

Visual identity for every remote tab

One window, zero sprawl

A single dock icon with horizontal and vertical tab bars. Every SSH session is a named, color-coded tab instead of an anonymous terminal window.

Identity bar per tab

Tab name, language version (Node / Python), virtualenv, git branch, framework, safety badge (DEV / STAGING / PROD / ROOT / SSH), and remote shell PID — all at a glance.

Auto-tab naming

Tabs that don't have a user label default to "Tab A", "Tab B", "Tab C" instead of a useless shell path. Rename any time.

Activity pulses

A subtle pulse on the tab color tells you which tabs are emitting output right now — active and background alike.

AI agent awareness

Agent auto-detection

Claude Code, Codex CLI, Gemini CLI, and Aider are detected from terminal output. The tab gets a color gradient, an agent badge in the identity bar, and its right-panel fills with that agent's custom commands and sub-agents.

/fork for conversations

Type /fork in any Claude, Codex, or Gemini session. A new tab opens with the same remote login and resumes the same conversation — two parallel branches diverging from your current point.

@tab: and @group:

Type @tab: in any terminal. A picker opens. Pick a tab. A formatted context block — identity, recent output, activity — is injected into your current prompt for the agent to read.

Custom commands travel with you

Drop .md files into Terminal Conductor's built-in Claude, Codex, and Gemini command folders — openable from Settings — and they sync to every SSH host on connect so your personal slash-commands work everywhere.

Session persistence that never lies

Remote tmux wrap

Every SSH tab is transparently wrapped in a named tmux session on the remote. Close the app, relaunch — your agent conversations and shells are still there, byte-for-byte.

Detached sessions

Close a tab with × and it detaches instead of dying. The caret next to + lists detached sessions; click to reattach.

Truly terminate

A red trash button beside Restart — two-click confirm — actually kills the remote tmux session and clears the database row. Use when you really mean it.

SSH auto-keepalive

ServerAliveInterval plus a null-byte ping keeps connections alive through NAT and firewall idle timeouts.

Know what's active on the remote

Remote environment probe

On connect, Terminal Conductor inspects the remote: Node/Python versions, virtualenv, framework (Next, Vite, Rails, Django, Express, and more), git branch and dirty state, NODE_ENV / RAILS_ENV / etc. Badges populate automatically.

Safety level inference

If any env var contains "prod", the tab is flagged PROD. Root logins are flagged ROOT. Staging and dev too. Never guess which host you're typing into.

In-flight process strip

A bottom-panel strip shows the foreground process and any services running on the remote — refreshed every 2 seconds from ps.

GitHub PR row

If the remote has gh authenticated, your current branch's PR status appears live in the bottom panel — check number, title, approvals, CI state.

Workspace organization

Workspace Overview

A grid view of every tab as a rich status card: activity glow, agent badge, branch, recent output preview, fork lineage. Click to jump; drag to regroup.

Manual groups

Name, color, and organize tabs into workspaces. Broadcast a command to a whole group. Generate a group briefing — a situation report of every tab's state in a new tab.

Split view

Right-click a tab → Split view → 2, 3, 4, or 5 columns. Tests, logs, agent, and editor side-by-side.

Pinned & Recent connections

Every host you connect to is remembered. Pin the ones you use constantly; the rest stay searchable in Recent.

Visual modes

Eight tab skins and five visual modes, all switchable per-tab or per-group. Your choice:

  • 1992 — green-phosphor CRT with beveled chrome and scanlines
  • Modern — clean flat dark chrome
  • Bubblegum Princess — pink/purple pastel with sparkle ANSI
  • Neon City — synthwave magenta-and-cyan gradients
  • Coding Cowboy — wood and leather with serif cowboy chrome
  • JARVIS HUD — animated SVG HUD border reacting to agent activity

Privacy by design

Zero telemetry

No analytics, no beacons, no external servers. Terminal Conductor never talks to anything you didn't explicitly configure.

Password scrubbing

Passwords never land in Recent history, activity events, or scrollback. A two-phase scrubber guards every surface that records command lines.

Local-only storage

Session database, settings, saved credentials — all stored in your App Support container. Nothing leaves your Mac.

Sandboxed

Mac App Store distribution. Every filesystem and network permission is declared in entitlements and enforced by macOS.

Requirements

  • macOS 14 (Sonoma) or later
  • Apple Silicon or Intel
  • An SSH server you can connect to (password or key)
  • tmux on the remote for session persistence (optional — tabs work without it, just don't persist)
  • gh authenticated on the remote for the GitHub PR row (optional)

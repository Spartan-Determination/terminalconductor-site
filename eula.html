1. What This App Does (and Doesn't Do)

Terminal Conductor is a terminal wrapper. It spawns real shells on your machine using the same PTY interfaces as any other terminal emulator, and renders the output in a visual intelligence layer (identity, context, memory, orchestration). Any command you type into a Terminal Conductor tab — or that an AI agent running inside a Terminal Conductor tab types on your behalf — is executed by your operating system exactly as if you had typed it into a plain terminal. Terminal Conductor does not review, sanitize, simulate, or reverse commands. It does not know which commands are safe.
